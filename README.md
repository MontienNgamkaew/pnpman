# PNPMAN - ระบบบริหารจัดการโครงสร้างสถานศึกษาอาชีวศึกษาและภาระงาน
## (White-Label & Premium Personnel Profile Org Chart System)

> ระบบบริหารจัดการข้อมูลบุคลากร โครงสร้างองค์กร และบอร์ดจัดภาระงานแบบลากวาง (Interactive Drag & Drop Org Chart) ออกแบบเป็นสากลในรูปแบบ **White-Label** เพื่อให้วิทยาลัยการอาชีพ วิทยาลัยเทคนิค หรือสถาบันการศึกษาต่าง ๆ สามารถนำไปปรับแต่งชื่อ โลโก้ และเลือกใช้ธีมสีระบบแบบพรีเมียมได้อย่างอิสระโดยไม่ต้องเขียนโค้ดเพิ่มเติม พร้อมระบบพิมพ์รายงาน PDF แดชบอร์ดวิเคราะห์ และคลังโปรไฟล์รูปภาพประจำตัวบุคลากรอัจฉริยะ

## 🌟 ฟีเจอร์เด่นระดับพรีเมียม (Premium Features)

### 🎨 1. ระบบ White-Label & Live Theme Customization
- **แบรนด์ของวิทยาลัยตัวเอง:** ผู้ดูแลระบบสามารถเปลี่ยนชื่อวิทยาลัย และอัปโหลดไฟล์โลโก้สถาบันได้โดยตรงผ่านหน้าเว็บ
- **ระบบธีมสีพรีเมียม 6 แบบ:** เลือกเปลี่ยนสีสันธีมหลักของวิทยาลัยได้แบบ Real-time ผ่านหน้าจอตั้งค่า (ระบบสลับตัวแปรสี CSS Variables) เช่น แดงทับทิม, เขียวมรกต, น้ำเงินไพลิน, ม่วงอเมทิสต์, ส้มแอมเบอร์ และชมพูโรสโกลด์
- **ปรับแต่งโครงสร้างได้อิสระ:** สามารถ จัดการฝ่ายงาน (`departments`) และงาน/ตำแหน่งย่อย (`jobs`) ทั้งเพิ่ม แก้ไข ลบ และเปลี่ยนลำดับการแสดงผล เพื่อรองรับความแตกต่างของโครงสร้างในแต่ละสถาบัน

### 🏗️ 2. บอร์ดลากวางโครงสร้างภาระงาน (Interactive Drag & Drop)
- **ลากวางแบบเรียลไทม์:** ลากรายชื่อครูและเจ้าหน้าที่จากคลังเข้าสู่ภารกิจต่าง ๆ (หัวหน้างาน, ผู้ช่วย, เจ้าหน้าที่, ครูในแผนกวิชา) ด้วย React Beautiful Dnd ลื่นไหล ไม่มีสะดุด
- **กฎการป้องกันตำแหน่ง (Role Rules Validation):** ควบคุมและตรวจสอบไม่ให้จัดตำแหน่งผิดสายงาน เช่น บังคับตำแหน่งหัวหน้างานได้สูงสุด 1 คนต่อจุด และกำหนดขอบเขตสายงานของเจ้าหน้าที่/ครูพิเศษ
- **รองรับหลายปีการศึกษา:** สลับเรียกดูย้อนหลังหรือล่วงหน้าได้ พร้อมปุ่ม **"คัดลอกจากปีก่อน"** เพื่อคัดลอกข้อมูลบอร์ดมาใช้ใหม่ในปีการศึกษาถัดไปได้ทันทีในคลิกเดียว

### 👥 3. ระบบโปรไฟล์และรูปภาพประจำตัว (Personnel Profile & Photos)
- **ช่องอัปโหลดรูปภาพทรงกลม:** อัปโหลดรูปภาพประจำตัวใบหน้าบุคลากรผ่านอินเตอร์เฟสรูปวงกลม รองรับสัดส่วนรูปถ่ายทางการได้อย่างดี พร้อมระบบ Preview ภาพทันที
- **ระบบจัดเก็บไฟล์อัจฉริยะ (Dynamic Cleanup):** หลังบ้านจะจำกัดขนาดไฟล์อัปโหลดรูปไม่เกิน 2MB รองรับเฉพาะสกุลไฟล์ที่ปลอดภัย (JPG, JPEG, PNG, SVG) และลบไฟล์รูปเก่าออกจากดิสก์เซิร์ฟเวอร์โดยอัตโนมัติทันทีเมื่ออัปเดตภาพใหม่หรือลบบุคลากร ป้องกันขยะล้นดิสก์
- **Avatar สีพาสเทลสำรอง:** กรณีที่บุคลากรยังไม่ได้อัปโหลดรูปถ่าย ระบบจะแสดงภาพไอคอนบุคคล (`User` หรือ `GraduationCap`) ภายใต้กรอบวงกลมสีพาสเทลที่แยกตามประเภทสายงานโดยอัตโนมัติอย่างประณีต

### ⚡ 4. ระบบการเรียงลำดับการ์ดแบบอัจฉริยะ (Smart Sorting)
- **เรียงลำดับตามสายงานอัตโนมัติ:** เมื่อวางบุคลากรในงาน การ์ดจะเรียงลำดับความสำคัญตามสายงานทันที: **ข้าราชการครู ➔ พนักงานราชการ ➔ ครูพิเศษสอน ➔ เจ้าหน้าที่**
- **สลับการ์ดแบบแมนนวล (Manual Reorder):** หากวิทยาลัยต้องการสลับตำแหน่งการ์ดตามความจำเป็นเฉพาะหน้า สามารถลากสลับลำดับการ์ดบนรายการเดียวกันได้ด้วยตนเองอย่างอิสระ ระบบจะบันทึกลำดับจัดเรียงนี้ถาวรลงฐานข้อมูลทันที

### 📊 5. บอร์ดสถิติและการวิเคราะห์เชิงลึก (Vacancy Analytics)
- แดชบอร์ดสรุปยอดรวมบุคลากรที่สำคัญ: ยอดรวม, จัดวางแล้ว, ยังไม่ได้จัดวาง, อัตราเฉลี่ยเจ้าหน้าที่ต่องาน
- **ระบบเรดาร์หาตำแหน่งว่าง (Vacancy Status):** วิเคราะห์และแจ้งเตือนทันทีหากมีงานย่อยหรือแผนกวิชาใดที่ไม่มีผู้ดำรงตำแหน่งหัวหน้างานบริหาร/หัวหน้าแผนก เพื่อป้องกันความผิดพลาดในการมอบหมายงาน

### 📄 6. ระบบส่งออกและพิมพ์รายงานระดับมืออาชีพ (Print & PDF)
- **Print to PDF:** สั่งพิมพ์รายงานบอร์ดโครงสร้างทั้งหมดแบบ A4 แนวนอน (Landscape) เพื่อนำไปจัดทำรูปเล่มประเมินวิทยาลัย โดยโลโก้, ชื่อวิทยาลัย, โทนสี และลำดับชื่อบุคลากรจะเปลี่ยนตามฐานข้อมูลจริง 100%
- **Excel (.xlsx) Export:** ส่งออกข้อมูลโครงสร้างในบอร์ดและรายชื่อบุคลากรแยกตามประเภทออกเป็นไฟล์สเปรดชีตแบบ 2 แท็บชีทสำหรับฝ่ายบริหาร

## 🚀 ขั้นตอนการติดตั้งและรันระบบ (Installation Guide)

### ความต้องการของระบบ (Prerequisites)
- **XAMPP** (แนะนำแพ็คเกจที่มี Apache + PHP 8.0 ขึ้นไป และ MySQL)
- **Node.js** รุ่น LTS (สำหรับผู้ที่ต้องการคอมไพล์หรือบิลด์โค้ดหน้าบ้านใหม่)

### ขั้นตอนการตั้งค่าระบบ:

#### 1. คัดลอกโปรเจกต์ไปที่โฟลเดอร์ XAMPP
นำโฟลเดอร์โปรเจกต์ `pnpman` ไปวางไว้ที่โฟลเดอร์ย่อย `htdocs` ของโปรแกรม XAMPP
```bash
# ตัวอย่างเส้นทางบน Windows:
C:\xampp\htdocs\pnpman\
```

#### 2. นำเข้าฐานข้อมูลและรัน Migrations
1. เปิดเว็บบราวเซอร์ ไปที่หน้าจัดการฐานข้อมูล `http://localhost/phpmyadmin/`
2. สร้างฐานข้อมูลใหม่ชื่อว่า `pnpman_db` (ตั้งค่า Collation เป็น `utf8mb4_unicode_ci`)
3. คลิกแท็บ **Import** และนำเข้าไฟล์ `database.sql` ที่อยู่ในโฟลเดอร์โปรเจกต์
4. รันระบบ Migration ทั้งหมดเพื่ออัปเดตระบบฐานข้อมูลให้รองรับฟีเจอร์ล่าสุด โดยเปิด URL ต่อไปนี้ตามลำดับในบราวเซอร์:
   - `http://localhost/pnpman/migrate_v2.php` (สร้างระบบตั้งค่าวิทยาลัยและธีมสี)
   - `http://localhost/pnpman/migrate_v3.php` (สร้างคอลัมน์จัดเรียงการ์ดแบบแมนนวล)
   - `http://localhost/pnpman/migrate_v4.php` (สร้างคอลัมน์เก็บภาพประจำตัวบุคลากร)

#### 3. ปรับตั้งค่าการเชื่อมต่อฐานข้อมูล
หากฐานข้อมูลของวิทยาลัยมีการใช้รหัสผ่านหรือเปลี่ยนชื่อ Host ให้แก้ไขการเชื่อมต่อในไฟล์ `api/db.php`:
```php
$host = 'localhost';
$dbname = 'pnpman_db';
$username = 'root';
$password = ''; // ปล่อยว่างไว้หากเป็นระบบ Default ของ XAMPP
```

#### 4. เปิดใช้งานหน้าเว็บหลัก
หลังจากติดตั้งฐานข้อมูลเรียบร้อยแล้ว สามารถเริ่มใช้งานโปรแกรมได้ทันทีผ่านเว็บบราวเซอร์ของคุณ:
👉 **เข้าสู่ระบบใช้งานจริง:** `http://localhost/pnpman/frontend/dist/`
*(หรือใช้ `http://localhost/pnpman/` ขึ้นอยู่กับโครงสร้างทางเข้าของโฮสต์จริง)*

*(สำหรับนักพัฒนาที่ต้องการแก้ไขส่วนหน้าบ้าน (React):)*
```bash
cd frontend
npm install
npm run dev    # รันบน Development server
npm run build  # คอมไพล์โปรเจกต์เพื่อส่งขึ้น Production dist/
```

1. **ลดงานด้านบุคลากรและแอดมิน 90%:** หลุดพ้นจากการนั่งจัดกล่อง จัดช่อง วาดเส้นบอร์ดโครงสร้างใน Photoshop หรือ PowerPoint ทุกครั้งที่มีการปรับตำแหน่ง ลากวางเสร็จในเสี้ยววินาที ระบบผังออนไลน์อัปเดตแบบเรียลไทม์ทันที
2. **พิมพ์รายงานส่งประเมินโรงเรียนได้เป๊ะรวดเร็ว:** กดพิมพ์ผังออกมาทาง PDF จัดรูปแบบพิมพ์แนวนอน (A4 Landscape) สวยสะดุดตา โลโก้และชื่อสถาบันถูกต้อง พร้อมรายนามบุคลากรตามสายงานที่จัดเรียงถูกต้องตามระเบียบ
3. **ไม่มีรายเดือน ข้อมูลอยู่กับตัว 100%:** รันบนเซิร์ฟเวอร์หรือ XAMPP ภายในวิทยาลัย ทำให้สามารถเก็บข้อมูลความลับของวิทยาลัย เช่น รูปถ่ายหน้าตรงไว้ภายในได้โดยไม่ต้องส่งออกภายนอกสถาบัน ฟรีค่าไลเซนส์ระบบตลอดการใช้งาน
4. **ภาพลักษณ์วิทยาลัยทันสมัยสไตล์ดิจิทัล:** หน้าเว็บดีไซน์พรีเมียม สไตล์กระจกฝ้าแก้ว (Glassmorphic Interface) สลับธีมสีและชื่อโลโก้จริงของสถาบันได้ทันที ช่วยสร้างความน่าเชื่อถือและความทันสมัยระดับมืออาชีพเวลาใช้ร่วมกับหน้าจอโปรเจกเตอร์หรือทีวีสรุปงานวิทยาลัย
